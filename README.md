# tokio-socks

Asynchronous SOCKS proxy support for Rust.

## Features

- [x] `CONNECT` command
- [ ] `BIND` command
- [ ] `ASSOCIATE` command
- [x] Username/password authentication
- [ ] GSSAPI authentication
- [ ] Asynchronous DNS resolution
- [ ] Chain proxies
- [ ] SOCKS4
